# ITMOParse

Парсер конкурсных списков поступающих в Университет ИТМО.

Проект автоматически получает актуальные данные через API приёмной комиссии ИТМО, обрабатывает информацию об абитуриентах и сохраняет её в удобном формате Excel для дальнейшего анализа.

## Возможности

- 📥 Получение списка образовательных программ
- 🔍 Загрузка конкурсных списков по выбранному направлению
- 👤 Парсинг информации об абитуриентах
- 📊 Сохранение результатов в Excel
- 🏆 Обработка конкурсных баллов
- 📚 Поддержка различных форм обучения и уровней образования

## Используемые технологии

- Python 3.11+
- requests
- openpyxl

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/Kwyro/ITMOParse.git
cd ITMOParse
```

Создайте виртуальное окружение:

### Linux/macOS

```bash
python -m venv .venv
source .venv/bin/activate
```

### Windows

```powershell
python -m venv .venv
.venv\Scripts\activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python src/main.py
```

После выполнения программы результаты будут сохранены в Excel-файл.

## Получаемые данные

Для каждого абитуриента могут быть получены:

- ID
- сумма конкурсных баллов
- баллы по предметам
- приоритет
- наличие оригинала документа
- особые права
- индивидуальные достижения
- другие сведения, доступные через API

## Формат результата

Результаты сохраняются в формате `.xlsx`, что позволяет:

- фильтровать данные;
- сортировать конкурсные списки;
- анализировать проходные баллы;
- отслеживать изменения конкурса.

## Использование

Проект может быть полезен:

- абитуриентам;
- родителям;
- разработчикам сервисов анализа поступления;
- всем, кто хочет автоматизировать обработку конкурсных списков.

## Важно

Данные загружаются из открытого API приёмной комиссии ИТМО. Структура API может изменяться, поэтому при изменениях на стороне университета может потребоваться обновление проекта.
